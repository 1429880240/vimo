# Vimo
---

> 一套针对HyBrid平台开发的H5框架

## 框架解决的问题

使用一套H5代码在各类HyBrid平台中运行, 并根据平台完成对应的初始化工作, 且能根据平台自动切换主题样式.

可以使用的平台:  **微信、支付宝、钉钉、普通浏览器、原生App内部增强版WebView中**.

## 如何开始

可以有两种方式使用: 

### npm安装
```
npm install vimo
```

### 使用startKit(推荐)

- [vimo-start-kit](https://github.com/DTFE/vimo-start-kit)

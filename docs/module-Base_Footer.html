Base/Footer

Header和Footer组件结构类似, 都是提供一个包裹容器, 不同的是一个固定在上面, 一个固定在下面.

Footer组件是Vimo页面的的三个主要构成之一, 主要是为Toolbar/Navbar/自定义结构提供一个容器,
该组件将始终固定在页面底部, Content组件会根据Footer的高度自动设定margin值.

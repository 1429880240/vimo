Component/Toggle

Toggle组件和Checkbox组件的功能类似, 但是Toggle组件在移动端更加好看, 也更加易用.

Toggle可以设置颜色, 当然不同模式下的样式还是不一样的, 感兴趣的可以切换试试.

Source:
  • toggle/toggle.vue, line 2
Properties:
Name Type Attributes Default Description
mode String <optional>
 'ios'

模式: "ios", "md", or "wp"
color String <optional>

颜色: "primary", "secondary", "danger", "light", and "dark"
checked Boolean <optional>
 false

选中状态, 一般用于初始值, 且无v-model的情况, 如果v-model和checked都使用的话, 则使用||选择
disabled Boolean <optional>
 false

禁用状态
